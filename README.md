# Heart Disease Prediction

## Objective
Build a machine learning model to predict the presence of heart disease based on patient health data.

## Dataset
- **Heart Disease UCI Dataset**
- **Source:** [Kaggle](https://www.kaggle.com/datasets/ronitf/heart-disease-uci)
- Includes clinical features such as:
  - Age
  - Cholesterol
  - Chest pain type
  - Maximum heart rate
  - ECG results
  - ST depression, and more

- The model achieved strong accuracy and demonstrated good class separation as shown by the ROC curve and confusion matrix.
- Key features influencing predictions included:
